CS2340 - Lab 1 (10 points)
Due 5 PM, Wednesday, September 8, 2010

Create a VB.NET program in MS Visual Studio 2010. The program has one form with a textbox, a label and a button.

The Form
  • The name is frmLab1.
  • The title is "Lab1" followed by your name.
  • The size of the form is (600, 400), the FormBorderStyle is Fixed3D.
  • The WindowState is Normal and the StartPosition is CenterScreen.
The Textbox
  • The name is txtName.
  • The textbox has a single line only.
  • The size is (150, 20).
  • It is centered both horizontally and vertically.
The Label
  • The name is lblName.
  • The AutoSize property is True.
  • The text of the label is "Enter Your Name."
  • It is above the textbox and horizontally centered.
The Button
  • The name is btnHello.
  • The text of the button is "Hello."
  • It is below the textbox and horizontally centered.
The Click Event of the button
  • Clicking on the button with no text (or spaces only) in the textbox, a message box with an OK button will be displayed with a message "Enter Your Name Please!" The MessageBoxIcon should be Exclamation and the Caption is "Lab 1."
  • Clicking on the button with any text containing at lease one non-space character in the textbox, a message box will be displayed with a different message. Assume the entered text is "CS 234" after spaces being removed from both ends, then the message should be 
         Hello, CS 234!
    The MessageBoxIcon should be Information and the Caption is "Lab 1."
