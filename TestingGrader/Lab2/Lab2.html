CS 2340 - Lab 2 (15 points)
Due 9 PM, Friday, September 14, 2012

Create a VB.NET program in MS Visual Studio 2010. The program has one form with 6 textboxes, 7 labels and 3 buttons.

The Form
  • The form name is frmLab2 and the title is "Lab2" followed by your full name.
  • The size of the form is (500, 450), the FormBorderStyle is FixedSingle, and the form has no ControlBox.
  • The WindowState is Normal and the StartPosition is CenterScreen.
The PayRoll Label
  • The name is lblPayRoll and the AutoSize property is True.
  • The text of the label is "PayRoll Program." The font size is 14, font name is Times New Roman, and font style is underlined.
  • It is above other controls and horizontally centered in the form.
The Six Pairs of Textboxes and Labels
  • Below the PayRoll label, there are three textboxes on the left and three textboxes on the right. All textboxes have the default size, and there a label above each textbox.
  • The names of the textboxes are txtID, txtRate, txtHours, txtGrossPay, txtDeduction, and txtNetPay.
  • The three textboxes on the left are displayed top-down, aligned on the left with the same vertical distance between them. The same applies to the three textboxes on the right, and each one on the right is aligned on the top with one textbox on the left.
  • The names of the labels above the textboxes are lblID, lblRate, lblHours, lblGrossPay, lblDeduction, and lblNetPay. The texts of the labels are "ID," "Rate," "Hours," "Gross Pay," "Deduction," and "Net Pay."
  • Each label is aligned on the left with the textboxe below it.
  • All the labels have the default font with the TextAligh being MiddleLeft .
  • The entier group of controls are centered in the form both horizontally and vertically.
  • The three textboxes on the left are used to get input values.
    • TextAlign is Left.
  • The three textboxes on the right are used to display output values.
    • TextAlign is Right.
    • All values must be in Currency format.
    • All are ReadOnly.
    • BackColor is Window, same as those input textboxes.
The Buttons
  • The names of the buttons are btnCompute, btnReset and btnExit.
  • The texts of the buttons are "COMPUTE," "RESET," and "EXIT."
    • The underlined characters are the access keys.
  • The three buttons are horizontally centered in the form with the same horizontally distance between them.
  • Button COMPUTE is the AcceptButton and button EXIT is the CancelButton of the form.
Tab Order
  • Textbox txtID has the focus when the program starts.
  • When the Tab key is pressed, the focus will go to txtRate, txtHours, btnCompute, btnReset, btnExit, and then back to txtID.
  • The output textboxes on the right will not receive focus when the Tab key is pressed.
The Click Event of btnCompute
  • The program will check the input values when btnCompute is clicked.
  • The values will be checked top-down, and when one invalid value is found, the remaining will not be checked.
  • When one input value is found invalid, the focus will go to the corresponding textbox so the user can enter the input again.
  • The ID must have at least one non-space character. Otherwise, a message box will be displayed with the following message: 
          Enter the ID please!
  • Rate must be a positive number. Otherwise, a message box will be displayed with one of the two following messages on two lines: 
          Invalid Rate!
      Rate must be a number!

      Invalid Rate!
      Rate must be positive!
  • Hours must be non-negative number. Otherwise, a message box will be displayed with one of the two following messages on two lines: 
          Invalid Hours!
      Hours must be a number!

      Invalid Hours!
      Hours must be non-negative!
  • A string in scientific notation such as "2e3" will be considered as an invalid number.
  • If all input values are valid, the program will compute the Gross Pay, Deduction, and Net Pay.
  • The Gross Pay is the product of Hours and Rate when Hours is at most 40.
  • Any hours above 40 are overtime hours and are paid one time and half of the regular rate.
  • The Deduction is 33% of the Gross Pay, and the Net Pay is the difference of the Gross Pay and the Deduction.
  • All output values must be displayed in the currency format.
The Click Event of btnReset
  • The program will clear all textboxes and move the focus to txtID when btnReset is clicked.
The Click Event of btnExit
  • The program will terminate when btnExit is clicked.
