The Grader Program from Last Year

  • The program is a console application.
  • When running, VBGrader.exe will grade the first exe file in the same folder which is not VBGrader.exe or *.vshost.exe.
  • For example, it will grade frmLab1.exe when running in the bin\debug folder, and grade YangQ_Lab1.exe when running in folder TestingGrader\Lab1.
  • The program uses an input file *.in and produces an output file *.out of the same name.
  • You can change the input file and see the different output file.
